## 游戏名

**《八方游侠》**

与 UI 中的"游侠阁"配套——玩家是游侠阁阁主，八方游侠来投。

## 世界框架：大晟王朝

### 世界概貌

朝代大晟（晟=光明兴盛），审美基调为唐宋之交。立国两百年，表面太平，暗面已烂：

- 朝廷腐败，地方豪强割据
- 边疆战事不断，溃兵流寇遍地
- 江湖门派各自为政
- 上古封印开始松动

方术不是西幻魔法，而是道家丹鼎派+阴阳家数术——画符、炼丹、望气、布阵，是对天地之力的运用。

### 三层冲突

1. **表层：乱世出英雄** — 各地不太平，玩家带队平事打响名号
2. **中层：朝堂与江湖** — 朝廷想收编江湖，江湖不想被管，玩家夹在中间
3. **深层：古物觉醒** — 上古封印松动，异象频出（长线暗线）

### 玩家角色：游侠阁阁主

游侠阁是江湖中介组织，不是传统门派。各地散人游侠挂名接活，阁主负责找活、分活、管后勤。

与玩法的对应：
- 招募 = 新游侠来挂单
- 编队 = 挑出活人选
- 关卡 = 接的委托
