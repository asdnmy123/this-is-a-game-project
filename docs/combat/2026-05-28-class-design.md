日期：2026-05-28
状态：approved

## 一、设计总则

### 角色定位三角

每个职业在"输出 - 生存 - 辅助"三条轴上有明确的主次定位，保证一上场就知道这角色干嘛的：

```
        输出
        /|\
       / | \
      /  |  \
惊鸿 --+-- 生存   (玄甲)
      \  |  /
       \ | /
        \|/
       辅助 (素问)
```

- **玄甲**：生存 >> 辅助 > 输出（站得住是第一要务）
- **方士**：输出 >> 辅助 > 生存（暴力法系，带控场）
- **惊鸿**：输出 >>> 辅助 > 生存（纯点杀机器）
- **素问**：辅助 >>> 生存 > 输出（奶就完事了）

### 技能数量约定

每个角色固定拥有：

| 技能类型 | 数量 | 说明 |
|---------|------|------|
| 普攻 | 1 | 每回合自动使用，无冷却，可选目标 |
| 主动技 | 2 | 玩家手动点击释放，有冷却（2~4 轮） |
| 被动技 | 1 | 自动触发，无需操作 |

共计 4 个技能/角色，3 角色上场 = 6 个主动按键 + 3 个普攻自动 + 3 个被动自动。操作门槛低，手机屏幕够放。

### 星级差异说明

四星和五星角色的差异**不体现在技能数量上**——都是 1 普攻 + 2 主动 + 1 被动。差异体现在：

- **属性成长曲线**（五星基础属性更高，每级成长更多）
- **被动效果强度**（如玄甲的被动：四星"血量低于 30% 时 DEF +15%"，五星"DEF +20% 且每轮回复 3% HP"）
- **技能数值**（五星技能的倍率/效果数值小幅领先）

星级不影响技能机制本身，只影响数值。这个原则在后续角色设计时用到。

### 技能资源系统说明

本方案所有技能仅受**冷却轮数**限制，没有 MP/怒气/能量系统。理由：

- **简洁优先**：少一套数值系统，减少实现和调优工作量
- **决策聚焦**：玩家决策从"省不省蓝"变为"什么时候放最合适"——时机选择替代资源管理
- **平台适配**：微信小游戏用户习惯轻量化操作，资源管理对休闲玩家门槛偏高

代价是技能之间的取舍主要靠冷却错开，设计时需要更精细地控制冷却和效果强度。

## 二、四大职业总览

| 条目 | 玄甲 | 方士 | 惊鸿 | 素问 |
|------|------|------|------|------|
| 一句话定位 | 铁壁城墙，守护队友 | 方术破敌，掌控战场 | 一箭封喉，逐个击破 | 悬壶济世，起死回生 |
| 定位三角 | 生存 >> 辅助 > 输出 | 输出 >> 辅助 > 生存 | 输出 >>> 辅助 > 生存 | 辅助 >>> 生存 > 输出 |
| 核心特色 | 技能吃 DEF（堆防即有输出） | 30% 防御穿透 | 高暴击（15% 基础） | 治疗 + 增益 |

## 三、玄甲（坦克）

### 3.1 职业定位

**铁壁城墙，守护队友。** 玄甲在前线吸收伤害，靠嘲讽和护盾让敌人打不动队友。输出不是他的活，但他盾击一下也不容忽视。

游侠阁里玄甲多是退役边军、镖师出身（世界观设定）。实战中玄甲是第一仇恨吸收者，是团队的定心丸。

核心玩法体验："你们打不动我队友，因为我在。" 玩家操作玄甲时，决策重点是——什么时候开嘲讽保队友，什么时候开减伤保自己。

### 3.2 核心机制：以守为攻

玄甲有两层核心机制：

**A. 技能吃 DEF（代替 ATK）**
- 玄甲的部分技能（含普攻）以 DEF 代替 ATK 计算伤害
- 公式：`技能伤害 = final_DEF × skill_multiplier / 100`（跳过 Step 1 的 ATK 部分）
- 后续流程不变（Step 2 ~ Step 8 正常走）
- 目的：玄甲堆防御不浪费，每点 DEF 既扛打又能打，避免"纯肉装只能挨打"的无聊感

**B. 仇恨核心**
- 嘲讽类技能强制敌方单位在持续回合内以玄甲为攻击目标
- 嘲讽持续期间，目标的攻击动作（普攻 + 单体技能）只能选择玄甲
- AOE 不受嘲讽影响（但 AOE 本来就会打到所有人）
- 具体机制见仇恨系统设计（后续单独文档）

### 3.3 属性偏向

| 属性 | 倾向 | Lv40 参考值（四星） | 说明 |
|------|------|-------------------|------|
| HP | 极高 | 4800 | 全职业最高，能挨最多下 |
| ATK | 低 | 320 | 攻击属性最差，普攻刮痧 |
| DEF | 极高 | 400 | 全职业最高，利用 DEF 做伤害的核心 |
| SPD | 慢 | ~80 | 无需先手，嘲讽在敌人出手前生效即可 |
| CRIT | 3% | 3% | 不堆暴击 |
| CRIT_DMG | +50% | +50% | 标准值 |

### 3.4 技能体系

| 技能 | 类型 | 效果概述 | 设计意图 |
|------|------|---------|---------|
| 盾击 | 普攻 | DEF×80% 单体伤害 | 让玄甲的普攻也吃 DEF，不至于普攻完全无用 |
| 铁壁嘲讽 | 主动 | 嘲讽全体敌人（持续 2 轮），自身 DEF +20%（持续 2 轮），冷却 4 轮 | 核心保护技能，开完嘲讽全队安心 |
| 护盾援护 | 主动 | 给一名队友施加护盾（吸收量 = DEF×150%），持续 2 轮，冷却 3 轮 | 单体保人，队友残血时用 |
| 百战之躯 | 被动 | 每损失 10% HP，受伤减免 +3%（最多 +30%，以受到伤害前的 HP 比例计算） | 越残越硬，符合"老兵"人设 |

> 注：以上为技能设计方向，具体数值和机制在后续"技能系统设计"阶段细化。

### 3.5 设计理由

**为什么技能吃 DEF 而不是给一个独立嘲讽值系统？**
- 简单。不需要引入第二套属性，DEF 本身就是玄甲的核心属性
- 自洽。堆防御 = 又硬又有输出，属性利用率高，不浪费
- 不破坏伤害公式。只是 Step 1 的 ATK 换成 DEF，后续全部复用

**为什么有两层保命（嘲讽 + 受伤减伤）？**
- 嘲讽保证敌人不打队友，是第一层保护
- 受伤减伤保证玄甲被集火时不会太快倒，是第二层保护
- 两层配合形成清晰的保人逻辑：嘲讽拉怪，减伤扛揍

**为什么 SPD 这么慢？**
- 嘲讽和护盾都是"施加后持续若干轮"的效果，不需要第一个出手
- 慢速玄甲作为兜底：等队友输出完了，玄甲才动，嘲讽覆盖下一轮敌人
- 速度慢也是坦克的常见特征，符合直觉

**为什么护盾吸收量是 DEF×150%（较高倍率）？**
- 玄甲 ATK 低，如果护盾吸收量用 ATK 或者某个固定值，后期会看起来很小
- 用 DEF 做吸收量，和玄甲的核心属性一致，成长性好
- 护盾有 3 轮冷却，吸收量高一点才值得用

## 四、方士（法师）

### 4.1 职业定位

**方术破敌，掌控战场。** 方士是团队的主要 AOE 输出 + 控场手，靠穿透无视敌人的高额防御。身板脆，但伤害高——正所谓高风险高回报。

游侠阁里的方士是道士、隐士出身（世界观设定），讲究"以术克刚"。战斗中方士负责压低全队血线或给敌人挂上 debuff，为惊鸿的点杀创造机会。

核心玩法体验："我出手，对面一片倒。" 玩家操作方士时，决策重点是——什么时候打 AOE 压血线，什么时候上 debuff 辅助，什么时候用单体补刀。

### 4.2 核心机制：穿透与控场

**A. 防御穿透 30%**
- 已定稿伤害公式中，方士的 `effective_DEF = target_DEF × 70%`
- 目标越硬，穿透收益越高（详见伤害公式文档 4.3 节）
- 法克坦，天然克制玄甲

**B. 控场技能**
- 方士拥有本作主要的 debuff 施加能力（减速、减攻、易伤等）
- 但控场技能伤害倍率低于纯输出技能，体现"控制有代价"
- Debuff 具体类型在后续技能系统设计中详细定义

### 4.3 属性偏向

| 属性 | 倾向 | Lv40 参考值（四星） | 说明 |
|------|------|-------------------|------|
| HP | 极低 | 2800 | 全职业最脆，被摸到很痛 |
| ATK | 高 | 500 | 高面板攻击配合穿透确保伤害，低于惊鸿倍率补偿后的单体期望 |
| DEF | 低 | 200 | 身板脆，靠站位和队友保护 |
| SPD | 中 | ~120 | 中速，保证先于大部分敌人出手上 debuff/压血线 |
| CRIT | 5% | 5% | 低暴击，不依赖暴击，靠穿透稳定输出 |
| CRIT_DMG | +50% | +50% | 标准值 |

### 4.4 技能体系

| 技能 | 类型 | 效果概述 | 设计意图 |
|------|------|---------|---------|
| 符箭 | 普攻 | ATK×100% 单体伤害 | 标准的法系普攻，不特殊，就是基础伤害 |
| 烈焰符 | 主动 | ATK×130% 全体 AOE 伤害，冷却 3 轮 | 核心 AOE，压低全队血线，为惊鸿点杀铺垫 |
| 缚灵咒 | 主动 | ATK×180% 单体伤害 + 施加"减速"（SPD -20%，持续 2 轮），冷却 3 轮 | 单体控场，让高速敌人变慢，给我方争取先手 |
| 通灵 | 被动 | 击败敌人时，所有主动技能冷却 -1 轮（同一轮内多次击杀可多次触发） | 收割补偿，鼓励方士配合惊鸿压血线收人头 |

> 注：以上为技能设计方向，具体数值和机制在后续"技能系统设计"阶段细化。

### 4.5 设计理由

**为什么方士是唯一的 AOE 职业？**
- 四个职业的 AOE 能力需要差异：玄甲无 AOE（单体保护），惊鸿无 AOE（纯点杀），素问有群奶但无伤害 AOE
- 只有方士有伤害 AOE，保证清场是法师的专属舞台
- 玩家在面对多个小怪时，方士是优先选择

**为什么控场技能伤害倍率低于纯输出？**
- 缚灵咒 180% 单体带减速，烈焰符 130% AOE 无控
- 如果控场技能又有高伤害又有强控，其他技能就没人用了
- 控制有代价——这是回合制策略的基础原则

**为什么被动是击杀冷却缩减？**
- 鼓励方士补刀，而不是打完 AOE 就没事干
- 击杀减冷却 → 能更频繁放 AOE → 滚雪球，有爽感
- 但方士本身暴击率低，补刀需要队友先压血线，依赖配合

**为什么中速（120）？**
- 快到能在大部分敌人前出手，先上控场/debuff
- 但不如惊鸿快（150），防止方士抢先秒人打乱节奏
- 中速方士配合减速 debuff 可以有效改变出手顺序

## 五、惊鸿（弓手）

### 5.1 职业定位

**一箭封喉，逐个击破。** 惊鸿是团队的单体点杀担当，靠高暴击和高速度先手秒掉敌方关键目标，创造人数优势。她不需要 AOE，因为她的任务是一个一个解决。

游侠阁里的惊鸿多是猎户、绿林出身（世界观设定），飘逸灵动。战斗中惊鸿负责快速减员，先手打残/打死一个，形成以多打少的局面。

核心玩法体验："你血条呢？" 玩家操作惊鸿时，决策重点是——优先点哪个目标、什么时候开爆发、先手暴击触发追击的快感。

### 5.2 核心机制：暴击点杀

**A. 高暴击基础**
- 15% 基础暴击率（对比其他职业 3%~5%），装备/Buff 可继续提升
- +60% 暴击伤害（对比其他职业 +50%）
- 暴击判定成功后，单体伤害爆表

**B. 单体特化**
- 所有技能均为单体目标，无 AOE
- 技能倍率全职业最高（单体 200%~280%）
- 专注点杀，不擅长处理群战

### 5.3 属性偏向

| 属性 | 倾向 | Lv40 参考值（四星） | 说明 |
|------|------|-------------------|------|
| HP | 中 | 3400 | 不算脆也不算硬，够挨两下 |
| ATK | 高 | 440 | 仅次于方士，高于玄甲和素问 |
| DEF | 中 | 280 | 标准水平，不会被秒也不硬扛 |
| SPD | 快 | ~150 | 全职业最快，优先出手创造人数优势 |
| CRIT | 15% | 15% | 吊打其他职业，核心输出属性 |
| CRIT_DMG | +60% | +60% | 暴击后伤害更高 |

### 5.4 技能体系

| 技能 | 类型 | 效果概述 | 设计意图 |
|------|------|---------|---------|
| 速射 | 普攻 | ATK×100% 单体伤害，暴击率额外 +5% | 普攻也有机会暴击，手感好 |
| 破甲箭 | 主动 | ATK×220% 单体伤害，冷却 3 轮 | 核心单体爆发技，倍率高，打脆皮大概率直接秒 |
| 鹰眼 | 主动 | 自身 ATK +30%、暴击率 +20%（持续 2 轮），冷却 4 轮 | 爆发前奏，下一轮打爆发 combo |
| 追猎 | 被动 | 击败敌人后，立即对另一个随机敌人发动一次普攻（追击伤害 = 普攻 60%，每轮最多触发一次，不可连锁触发） | 击杀滚雪球，体现"逐个击破"的定位 |

> 注：以上为技能设计方向，具体数值和机制在后续"技能系统设计"阶段细化。

### 5.5 设计理由

**为什么惊鸿没有 AOE？**
- 四个职业有明确分工：方士 AOE 压血线，惊鸿单体点杀
- 如果惊鸿也能 AOE，方士的定位就被挤占了
- "点杀特化"是惊鸿的品牌标识

**为什么鹰眼（自 buff）冷却比破甲箭长？**
- 鹰眼是组合技的核心组件：先鹰眼 → 下轮破甲箭触发高暴击高伤害
- 冷却 4 轮 > 破甲箭 3 轮，意味着不能每次破甲箭都配合鹰眼，需要选择时机
- 策略深度：什么时候开鹰眼、打谁，是惊鸿玩家的核心决策

**为什么追击只继承 60% 普攻伤害？**
- 如果 100% 继承，击杀后连续追击可能一回合清场，太强
- 60% 保证追击是个"添头"——补刀残血够了，但不会因为一次暴击击杀就滚雪球赢全场

**为什么高 SPD（150）？**
- 惊鸿的核心价值是"先手减员"：在敌人出手前先干掉一个，形成以多打少
- 如果惊鸿速度不够快，敌人先动了再杀就没意义了
- 高速 + 点杀 = 惊鸿的品牌定位

**为什么暴击率比别人高这么多（15% vs 3%~5%）？**
- 暴击是惊鸿的专属赛道
- 数值策划可以通过调整暴击率来精确控制惊鸿的期望输出
- 高暴击带来高方差（有时一刀秒，有时不暴击刮痧），这种刺激感符合弓箭手的职业幻想

## 六、素问（奶妈）

### 6.1 职业定位

**悬壶济世，起死回生。** 素问是团队的生命线，负责回复和增益。她不能打，但她能让队友一直活着。一个队伍里有没有素问，是两个游戏。

游侠阁里的素问是正统医道传人（世界观设定），江湖上谁都给三分面子。战斗中素问是团队的定心丸——只要素问站着，局势就有回转余地。

核心玩法体验："有我在，死不了。" 玩家操作素问时，决策重点是——预判谁接下来会掉血、大招是留着救急还是现在就用、要不要上 buff 抬一下输出。

### 6.2 核心机制：治疗与增益

**治疗机制（已定稿）：**
- `heal = final_ATK × heal_multiplier / 100`
- 不受目标 DEF 影响、不受等级补正、不受暴击影响（除非被动解锁）
- 受"治疗效果 +X%"修正
- 详见伤害公式文档第 8 章

**素问的特色：**
- 治疗走 ATK 乘区，所以素问需要堆适量的 ATK
- 素问是唯一的治疗职业（无人竞争）
- 部分技能可提供护盾或增益效果（加攻、加防等）

### 6.3 属性偏向

| 属性 | 倾向 | Lv40 参考值（四星） | 说明 |
|------|------|-------------------|------|
| HP | 中高 | 3600 | 仅次于玄甲，保证奶妈不会轻易倒 |
| ATK | 低 | 280 | 面板 ATK 最低，但治疗倍率高 + 治疗效果加成补偿 |
| DEF | 中 | 280 | 标准水平，比玄甲脆比法师硬 |
| SPD | 中 | ~100 | 中速，保证在队友掉血后能及时奶 |
| CRIT | 3% | 3% | 不堆暴击，除非被动解锁治疗暴击 |
| CRIT_DMG | +50% | +50% | 标准值 |

### 6.4 技能体系

| 技能 | 类型 | 效果概述 | 设计意图 |
|------|------|---------|---------|
| 金针 | 普攻 | ATK×100% 治疗最低血量队友 | 普攻也是奶，让素问每回合都有治疗贡献 |
| 回春术 | 主动 | ATK×320% 单体治疗 + 净化一个负面效果，冷却 3 轮 | 核心单奶，还能解 debuff，救急用 |
| 益气汤 | 主动 | ATK×180% 全体治疗，冷却 4 轮 | 群奶应对 AOE 场景，冷却长以防无脑群奶 |
| 悬壶 | 被动 | 每轮素问行动结束时，治疗全队 ATK×40% | 每回合白送微量群奶，维持血线健康 |

> 注：以上为技能设计方向，具体数值和机制在后续"技能系统设计"阶段细化。

### 6.5 设计理由

**为什么素问是唯一的治疗职业？**
- 这是刻意设计：治疗是全队的刚需，但不让"半奶"角色稀释素问的定位
- 队伍里要带素问还是不带素问硬拼，本身就是一个策略决策
- 如果多个职业都能奶，素问的独特价值就没有了

**为什么普攻是治疗而非伤害？**
- 素问 ATK 最低（280），打人 = 刮痧，玩家不会想用她打人
- 改成普攻奶最低血量队友，每回合都有用，手感好
- 自动战斗时素问自动奶残血，符合期望

**为什么益气汤冷却比回春术长？**
- 群奶（120%×3=360% 总治疗量）的总效率远高于单奶（200%）
- 如果冷却一样短，回春术就没人用了
- 4 轮冷却意味着群奶是"关键时刻用"，不是每轮都用的技能

**为什么被动是微量群奶？**
- 素问在什么也不做的时候（比如所有队友满血）不会完全闲着
- 30% ATK 治疗量很小（Lv40 约 84 血），不会影响平衡
- 维持血线健康，防止队友被 AOE 压低后下一波直接死

**为什么中速（100）？**
- 太快的素问可以在队友掉血前就奶满，失去了"预判治疗"的策略感
- 太慢的素问可能在队友死后才奶，体验很差
- 中速 + 普攻奶被动 = 一般队友先动掉血，然后素问奶起来，然后敌人再动

## 八、设计验证

### 8.1 职业区分度检查

| 对比组 | 核心区分 | 会不会混淆 |
|--------|---------|-----------|
| 玄甲 vs 方士 | 一个负责扛（堆 DEF），一个负责打（堆 ATK + 穿透） | 完全相反，不会混淆 |
| 玄甲 vs 惊鸿 | 玄甲慢速保人，惊鸿高速杀敌 | 定位完全不同 |
| 玄甲 vs 素问 | 玄甲扛伤害，素问奶伤害 | 一个防患未然，一个事后补救，互补 |
| 方士 vs 惊鸿 | 方士 AOE + 控场，惊鸿单体点杀 | 技能类型完全不同 |
| 方士 vs 素问 | 方士打人 + 上 debuff，素问奶人 + 上 buff | 一个搞对面，一个保自己 |
| 惊鸿 vs 素问 | 惊鸿输出，素问辅助 | 一个杀，一个救 |

结论：四个职业两两之间都有明确区分，不存在"XX 和 XX 差不多"的模糊情况。

### 8.2 与已定稿设计的一致性检查

| 已定稿设计 | 本方案遵循情况 | 说明 |
|-----------|--------------|------|
| 伤害公式（DEF_CONST=500） | 遵循 | 各职业属性分配与伤害公式验算表一致 |
| 方士穿透 30% | 遵循 | 方士核心机制 |
| 暴击率分配（玄甲 3%/方士 5%/惊鸿 15%/素问 3%） | 遵循 | 直接沿用 |
| 治疗公式（ATK×倍率，不受防御/暴击影响） | 遵循 | 素问技能基于此设计 |
| 回合流程（速度排序，每人一动） | 遵循 | 速度分级符合"一轮一动"规则 |
| 玩家先手软保底 | 遵循 | 惊鸿高速 + 软保底保证玩家至少有一人先动 |
| 世界观设定（大晟王朝/游侠阁） | 遵循 | 各职业的江湖来历与世界观一致 |
| 四数值设定（HP/ATK/DEF/SPD） | 遵循 | 属性偏向参考了 Lv40 验算表的数值 |

### 8.3 策略深度检查

三个位置的策略交互：

```
玄甲 ←→ 素问
  嘲讽保队友  ↔  治疗保玄甲
  套盾保脆皮  ↔  净化救急

玄甲 ←→ 惊鸿
  拉住仇恨让惊鸿安全输出  ↔  快速解决威胁目标减轻玄甲压力

方士 ←→ 惊鸿
  AOE 压血线  ↔  单体补刀击杀
  减速控场  ↔  高速先手

素问 ←→ 所有人
  全体治疗应对 AOE  ↔  队友承受伤害需要治疗
  增益 buff 提升输出  ↔  输出更高更快结束战斗
```

初步结论：
- 四大职业有清晰的配合链：玄甲扛 → 素问奶 → 方士压血线 → 惊鸿点杀
- 每个职业都有存在感，没有任何职业是"挂件"
- 可形成多种战术组合（双奶苟命流、双法 AOE 流、单核保大哥流等）
